# Análise de Logs de Autenticação

## Objetivo
Identificar tentativas de login falhadas a partir de arquivos de log, auxiliando na detecção de possíveis ataques de força bruta.

## Importância em segurança
A análise de logs é uma das principais atividades em um SOC, permitindo identificar padrões suspeitos e responder a incidentes de forma proativa.

## Exemplo de risco
Múltiplas tentativas falhadas vindas do mesmo endereço IP podem indicar ataque de força bruta ou credenciais comprometidas.

## Possíveis melhorias
- Identificação de IPs suspeitos
- Geração de alertas
- Integração com sistemas de monitoramento
