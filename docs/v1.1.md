# Security Automation Toolkit — v1.1 (Scope)

## Objetivo
Toolkit educacional em Python para automação de enumeração básica de rede e geração de relatório HTML com classificação de risco.

## O que a v1.1 entrega
- Varredura simples de host/portas
- Coleta e organização dos resultados
- Classificação de risco: low / medium / high
- Relatório final em HTML com resumo executivo

## O que NÃO entra na v1.1
- Exploração de vulnerabilidades
- Ataques de senha
- Enumeração avançada
